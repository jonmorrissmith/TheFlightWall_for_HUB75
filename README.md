# TheFlightWall for HUB75

If you're looking for the original repositing then it's [here on GitHub](https://github.com/AxisNimble/TheFlightWall_OSS) with a build video [here on Instagram](https://www.instagram.com/p/DLIbAtbJxPl)

# So What's going on here?

Porting the OpenSource version to use HUB75 LED-matrix displays by leveraging the [rpi-rgb-led-matrix library](https://github.com/hzeller/rpi-rgb-led-matrix).

See my [Raspberry Pi HUB75 Train Departure Board project for an example](https://github.com/jonmorrissmith/RPi_DepartureBoard)
